# KKBD-USB

## 概要
* SBC6800やKZ80マイコンなどUART経由で稼働するシングルボードコンピューターをスタンドアロン化する際のキーボードI/Fボードとしたい。
* github https://github.com/kuninet/KKBD-USB

## 仕様概要
* RaspberryPi PicoによりUSBキーボード入力をUART(9600bps)へ中継する。
* HW設定（ジャンパーピンによるハードウェア設定）にて以下を設定可能とする
  * 行末コードをCRLF/CR/LFを選択できるようにする。（JP1/JP2の2ピンで設定）
  * UARTのスピードを選択できるようにする。（JP3/JP4の2ピンで設定）
