# KKBD-USB

## 概要
* SBC6800やKZ80マイコンなどUART経由で稼働するシングルボードコンピューターをスタンドアロン化する際のキーボードI/Fボードとしたい。
* github https://github.com/kuninet/KKBD-USB

## 仕様概要
* PaspberryPi PicoによりUSNキーボード入力をUART(9600bps)へ中継する。
* HW設定にて以下を設定可能とする
  * 行末コードをCRLF/CR/LFを選択できるようにする。
  * UARTのスピードを選択できるようにする。
